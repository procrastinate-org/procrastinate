App.builtin_tasks
"""""""""""""""""
Builtin tasks are registered when the App object is initialized, and
are accessible through :py:attr:`App.builtin_tasks`:

.. code-block:: python

    app.builtin_tasks["remove_old_jobs"].defer(max_hours=72)
